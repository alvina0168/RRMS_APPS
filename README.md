# RRMS_APPS
RRMS is a simple Android application designed to help kindergarten teachers manage student registration, attendance, and class records efficiently. 
The app reduces manual paperwork and provides an easy-to-use digital system for daily classroom management.

# Technologies Used
Language: Kotlin

UI: Jetpack Compose

Database: Room Database

Architecture: MVVM (basic)

# How It Works
1. Teacher logs into the app
2. Students are registered using the registration form
3. Attendance is marked daily
4. Student records can be viewed, edited, or deleted
5. Summary information is displayed on the Home Screen
